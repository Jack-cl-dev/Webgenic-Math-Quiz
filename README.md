# WebGenic Math Quiz
## Language & information
The program is written in C#, as this was allowed in the program specifications and never disallowed anywhere else, and I personally prefer using it over python.

The IDE used was JetBrains Rider.

I have chosen to put this code on Github because it allows for much easier file uploads, and more importantly still, revisions (should they be necessary).

## Running the program
The quickest and easiest way to run the program is the EXE in the releases section. Otherwise, the code can be previewed here in Github. If you wish to compile and run it yourself, the program runs on DotNet 10, and you will need the DotNet 10 SDK installed. Some IDE's handle this mostly autonomously.
